迷人的下划线

FE炼金术研究所
2018-03-14
CSS


上学或是平时读书，在遇到重点语句时，常常想要做些标记来加深印象，也方便以后查找。下划线 无疑是大家最好的选择。
那如何利用HTML与CSS给文字添加下划线呢？
最广为人知的方法是以下三种：

  1. 使用<u>
  2. 将CSS中text-decoration属性设置为underline
  3. 利用border-bottom

下面展开来讲讲。
首先，第一种已经过时了，建议大家不要使用。偏重外观呈现的方面，我们本就该用CSS，而非HTML来解决。更别提HTML5规定说<u>的正确打开方式是用来标记 the Unarticulated Annotation ，即一些拼错的字词。配合CSS使用，可以获得word中字词检查发现错误时的效果：

后面两种倒是没多大差别，text-decoration与border都自带color与style属性，想要不同颜色与风格也是轻而易举。但如果你还想要设置 间距 与 厚度，那方法2就无能为力，只能依赖方法3了。
border本身可以设置width，还可以通过添加<span>元素设置padding-bottom的方法来添加间距。
